# Week 1 – Literature Review

**Duration:** Week 1

## Objective

Build a strong understanding of Post-Quantum Cryptography (PQC) implementation challenges on constrained IoT devices by studying existing research, benchmarking methodologies, and hardware limitations.

## Papers Reviewed

### 1. Constrained Device Performance Benchmarking with the Implementation of Post-Quantum Cryptography

- Studied benchmarking methodologies for evaluating PQC algorithms on resource-constrained devices.
- Analyzed performance metrics including:
  - Key Generation
  - Encapsulation
  - Decapsulation
  - Execution Time
  
- Understood factors affecting benchmark accuracy and reproducibility.

---

### 2. Secure IoT in the Era of Quantum Computers—Where Are the Bottlenecks?

- Explored security challenges introduced by quantum computing for IoT systems.
- Identified major deployment bottlenecks:
  - Limited computational resources
  - Memory constraints
  - Energy consumption
  - Communication overhead
- Reviewed suitable PQC approaches for constrained environments.

---

### 3. pqm4: Testing and Benchmarking NIST PQC on ARM Cortex-M4

- Studied the pqm4 framework for benchmarking NIST PQC algorithms.
- Learned about:
  - Benchmark setup
  - Performance measurement techniques
  - Cycle-count analysis
  - Memory evaluation
- Understood why Cortex-M4 is widely used as a reference platform for embedded PQC research.

---

## Presentation

Prepared and delivered presentations covering the three research papers, discussing:

- Research objectives
- Methodologies
- Experimental setup
- Performance metrics
- Key findings
- Limitations
- Relevance to our research

## Key Learnings

- Understood the current landscape of PQC implementation on constrained IoT devices.
- Learned the standard benchmarking metrics used in PQC research.
- Identified computational and memory bottlenecks in embedded systems.
- Gained familiarity with the pqm4 benchmarking framework.
- Established a foundation for future experimental benchmarking.

## Outcome

Successfully completed the literature review phase and developed a strong theoretical foundation for benchmarking Post-Quantum Cryptography algorithms on constrained devices.
